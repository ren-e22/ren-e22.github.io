    Comando para listar documentos
    El comando ls se utiliza para listar archivos o directorios en Linux
    El comando ls -l muestra todoa la informacion de los archivos como permisos usuario o grupos en Linux

    El comando cat puedes crear un archivo rápidamente
    Comando para crear un directorio
    El comando mkdir puedes crear un archivo rápidamente
    Comando Nano
    Coamndo nano nombre_archivo Esto abrirá una nueva ventana de editor y se puede comenzar a editar el archivo
    Con Ctrl q se cierra el archivo
